## 一、设计概述

### 1.1 核心概念

思维树图是 Little Ice 对话管理的可视化导航系统。它将对话组织成树形结构，让用户可以：
- 快速定位和切换对话
- 理清对话之间的逻辑关系（父子、分支）
- 在全局视角下浏览知识结构

### 1.2 设计目标

- **轻量**：不增加认知负担，树形图作为辅助导航存在
- **干净**：无过多装饰，信息元素克制
- **专注**：只读浏览模式，不用于编辑节点关系
- **一致**：与对话视图共用 Header，切换丝滑

## 四、树形图渲染

### 4.1 技术选型

| 项 | 选型 |
|----|------|
| 渲染引擎 | React Flow v11+ |
| 布局算法 | dagre（自上而下，TB 朝向） |
| 节点组件 | 自定义组件（非 HMTL 内置） |
| 连接线 | smoothstep（平滑贝塞尔曲线） |

### 4.2 布局配置

```typescript
const layoutConfig = {
  rankdir: 'TB',       // 自上而下
  align: 'UL',         // 左上对齐
  nodesep: 40,         // 同级节点间距
  ranksep: 100,        // 层级间距
  marginx: 40,         // 水平边距
  marginy: 40,         // 垂直边距
};
```

### 4.3 节点设计

**尺寸**：宽度 240px × 高度 72px

**布局结构**：
```
┌──────────────────────────────────┐
│ 📝 标题文字（最多显示一行）      │
│ 最后活跃：15分钟前 · 3条消息     │
└──────────────────────────────────┘
```

### 4.4 层级与图标

| 层级 | 图标 | 说明 |
|------|------|------|
| 根节点 | 📝 | 代表顶级对话主题 |
| 子节点（第2层） | 🔹 | 分支对话 |
| 子节点（第3层） | 🔸 | 细分主题 |
| 子节点（第4层+） | ○ | 更细粒度，仅用圆点 |

### 4.5 连接线

```typescript
const edgeStyle = {
  type: 'smoothstep',
  animated: false,
  style: {
    stroke: '#D1D5DB',
    strokeWidth: 1.5,
  },
};
```

**悬停高亮**：鼠标悬停在连接线上时，颜色加深至 `#9CA3AF`，宽度增至 2px

### 4.6 React Flow 配置

```jsx
<ReactFlow
  nodes={layoutedNodes}
  edges={edges}
  nodeTypes={customNodeTypes}
  fitView
  fitViewOptions={{ padding: 0.3 }}
  nodesDraggable={false}    // 禁用拖拽
  nodesConnectable={false}  // 禁用连线
  elementsSelectable={true} // 允许选中
  zoomOnScroll={false}      // 禁用缩放
  panOnDrag={false}         // 禁用平移
  preventScrolling={true}   // 阻止页面滚动
  minZoom={0.3}
  maxZoom={2}
  onlyRenderVisibleElements={nodeCount > 100}
  attributionPosition="bottom-left"
/>
```

## 六、悬停菜单

### 6.1 触发方式

鼠标悬停在节点上时，右上角显示三点图标（`⋮`），点击展开菜单。

```
┌──────────────────────────────────┐
│ 📝 需求分析              ⋮      │
│ 最后活跃：15分钟前 · 3条消息     │
└──────────────────────────────────┘
```

### 6.2 菜单项

| 操作 | 说明 |
|------|------|
| 查看对话 | 切换到对话视图并显示该节点内容 |
| 添加子节点 | 创建子节点并切换到对话视图 |
| 重命名 | 进入内联编辑模式 |

## 十、边界情况处理

### 10.1 空树

```
┌──────────────────────────────────┐
│  思维树图                        │
│                                  │
│        🌲 还没有任何节点        │
│        [创建第一个对话]          │
│                                  │
└──────────────────────────────────┘
```

### 10.2 单节点

提供「添加子节点」的引导，节点下方显示操作提示。

### 10.3 深度嵌套

- 默认最多展开 5 层
- 深层节点显示为「... (+N)」折叠状态
- 可手动展开

### 10.4 数量级性能策略

| 节点数量 | 渲染策略 | 悬停效果 |
|----------|----------|----------|
| ≤ 30 | 完整渲染 | 全开 |
| ≤ 100 | 完整渲染 | 关闭连线高亮 |
| ≤ 500 | 启用虚拟化 | 关闭 |
| > 500 | 默认收起子节点 | 关闭 |
